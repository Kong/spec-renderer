# SlideOut

SlideOut is an overlay container with backdrop that displays content over the page area.

## Props

### `visible`

* type: `boolean`
* required: `false`
* default: `false`

### `maxWidth`

* type: `string`
* required: `false`
* default: `500px`

Max width of SlideOut container.

## Slots

### `default`

Slot for SlideOut content.
